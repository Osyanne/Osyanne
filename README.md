## 🚀 Projects I maintain

### 🎵 [osu-mcp](https://github.com/Osyanne/osu-mcp)
[![PyPI](https://img.shields.io/pypi/v/osu-mcp.svg)](https://pypi.org/project/osu-mcp/) [![License](https://img.shields.io/badge/License-MIT-yellow.svg)](https://github.com/Osyanne/osu-mcp/blob/main/LICENSE)

MCP server that exposes the osu! API v2 as 12 tools for Claude Desktop and any MCP client. Player stats, score lookups, beatmap search, leaderboards — all in plain English. Published on the official **MCP Registry** as `io.github.Osyanne/osu-mcp`.

`Python` · `MCP` · `Anthropic SDK` · `osu! API`

### 🏙 [CS2 OSM Toolkit](https://github.com/Osyanne/CitiesSkylines2-osm-toolkit) ⭐ 16+
GIS pipeline that extracts real-world zoning data from OpenStreetMap and exports it for Cities: Skylines II. 10+ cities live (Minneapolis, Trondheim, Amsterdam, and more), browser-based generator on the way. Patreon-supported.

`Python` · `GIS` · `OpenStreetMap` · `Cities: Skylines II`
